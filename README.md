# Landing Page – Entre algoritmos y yo

Este es un sitio web simple para promocionar el libro "Entre algoritmos y yo" de Juan C. Pupo.

## Contiene:

- index.html (Landing page)
- portada.jpg (Portada del libro)
- capitulo3.pdf (Capítulo gratuito para descarga)
- Formulario conectado a Formspree

## ¿Cómo publicarlo en GitHub Pages?

1. Crea un repositorio en GitHub (ej. `entre-algoritmos-landing`)
2. Sube estos archivos
3. Ve a Settings > Pages
4. En "Source", selecciona la rama `main` y carpeta `/ (root)`
5. Tu sitio se publicará en: `https://TUUSUARIO.github.io/entre-algoritmos-landing/`
